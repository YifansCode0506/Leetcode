# 集合与位运算表示

## 前言

本文将集合论运算与编程语言中常见的位运算统一起来理解。

在编程、数学中集合（Set Theory）用来表示对象的无序集合。例如，假设全集下面的集合为 `{a, b, c}`，任意子集可以看作每个元素是否被选中的一种状态。例如 Java 中的 `HashSet`、C++ 中的 `std::unordered_set`。

在集合论中，有交集 ∩ 、并集 ∪、差集、补集等基本概念。如果用集合元素“是否存在”来表示状态，就可以用二进制位来对应集合中的元素。

设全集为 `{a, b, c}`，我们可以用 3 位二进制数表示它的所有子集：

| 子集 | 二进制 |
| --- | --- |
| `{}` | `000` |
| `{a}` | `001` |
| `{b}` | `010` |
| `{a, b}` | `011` |
| `{c}` | `100` |
| `{a, c}` | `101` |
| `{b, c}` | `110` |
| `{a, b, c}` | `111` |

第 `i` 位为 `1`，表示第 `i` 个元素在集合中；第 `i` 位为 `0`，表示第 `i` 个元素不在集合中。

如果集合有 `n` 个元素，那么共有 `2^n` 个子集。

利用位运算「并行计算」的特点，我们可以高效地做一些和集合有关的运算。按照常见的应用场景，可以分为以下四类:

1. 集合与集合
2. 集合与元素
3. 遍历集合
4. 枚举集合

########################################################################

## 一、集合与集合

其中 `A` 表示旧集合，`B` 表示新集合。`&` 表示按位与，`|` 表示按位或，`^` 表示按位异或，`~` 表示按位取反。

两个集合的「并、交、差、对称差」等关系，可以直接用位运算表达。

| 术语 | 集合 | 位运算 | 集合示例 | 位运算示例 |
| --- | --- | --- | --- | --- |
| 交集 | `A ∩ B` | `a & b` | `{0, 3, 5} ∩ {0, 1, 3} = {0, 3}` | `101001 & 001011 = 001001` |
| 并集 | `A ∪ B` | `a | b` | `{0, 3, 5} ∪ {0, 1, 3} = {0, 1, 3, 5}` | `101001 | 001011 = 101011` |
| 对称差 | `A △ B` | `a ^ b` | `{0, 3, 5} △ {0, 1, 3} = {1, 5}` | `101001 ^ 001011 = 100010` |
| 差 | `A \ B` | `a & ~b` | `{0, 3, 5} \ {0, 1, 3} = {5}` | `101001 & 110100 = 100000` |
| 差（子集） | `A \ B, B ⊆ A` | `a ^ b` | `{0, 1, 3} \ {0, 3} = {1}` | `001011 ^ 001001 = 000010` |
| 包含于 | `A ⊆ B` | `(a & b) == a` 或 `(a | b) == b` | `{0, 3} ⊆ {0, 1, 3}` | `001001 & 001011 = 001001` |

注意：

1. 按位取反时，不同语言的整数位宽不同。处理有限全集时，常用 `mask = (1 << n) - 1` 限制范围，再写作 `(~a) & mask`。
2. 判断 `A` 是否为 `B` 的子集，可以用 `(a & b) == a`。
3. 判断两个集合是否没有交集，可以用 `(a & b) == 0`。

## 二、集合与元素

通常把集合中的元素编号为 `0, 1, 2, ...`，第 `i` 个元素对应二进制的第 `i` 位。

其中 `s` 表示集合，`i` 表示元素编号。

| 术语 | 集合 | 位运算 | 集合示例 | 位运算示例 |
| --- | --- | --- | --- | --- |
| 空集 | `∅` | `0` | `{}` | `0` |
| 单元素集合 | `{i}` | `1 << i` | `{2}` | `1 << 2` |
| 全集 | `U = {0, 1, ..., n - 1}` | `(1 << n) - 1` | `{0, 1, 2, 3}` | `(1 << 4) - 1 = 1111` |
| 补集 | `U \ S` | `((1 << n) - 1) ^ s` | `U = {0, 1, 2, 3}`，`S = {0, 1}` | `1111 ^ 0011 = 1100` |
| 属于 | `i ∈ S` | `(s >> i) & 1 == 1` | `2 ∈ {0, 2, 3}` | `(1101 >> 2) & 1 = 1` |
| 不属于 | `i ∉ S` | `(s >> i) & 1 == 0` | `1 ∉ {0, 2, 3}` | `(1101 >> 1) & 1 = 0` |
| 添加元素 | `S ∪ {i}` | `s | (1 << i)` | `{0, 2} ∪ {1}` | `0101 | 0010 = 0111` |
| 删除元素 | `S \ {i}` | `s & ~(1 << i)` | `{0, 1, 3} \ {1}` | `1011 & 1101 = 1001` |
| 删除最小元素 | 去掉最低位的 `1` | `s & (s - 1)` | `{0, 1, 3}` 去掉 `0` | `1011 & 1010 = 1010` |
| 取最低位元素 | 只保留最低位的 `1` | `s & -s` | `{0, 1, 3}` 取 `0` | `1011 & 0101 = 0001` |

示例：

```text
          s = 101100
      s - 1 = 101011  // 最低位的 1 变成 0，右侧 0 变成 1
s & (s - 1) = 101000
```

特别地，假设 `s` 是正整数，那么 `s & (s - 1) == 0` 表示 `s` 是 2 的幂。

此外，编程语言提供了一些和二进制有关的库函数，例如:

1. 计算二进制中的1的个数，也就是集合大小;
2. 计算二进制长度，减一后符到集合最大元素;
3. 计算二进制尾零个数，也就是集合最小元素。

调用这些函数的时间复杂度都是O(1)。

| 语言 | 二进制长度 | 集合大小 / 1 的个数 | 前导零个数 | 后缀零个数 |
| --- | --- | --- | --- | --- |
| Python | `x.bit_length()` | `x.bit_count()` | - | - |
| Java | `Integer.toBinaryString(x).length()` | `Integer.bitCount(x)` | `Integer.numberOfLeadingZeros(x)` | `Integer.numberOfTrailingZeros(x)` |
| C++ | `bit_width(x)` | `popcount(x)` | `countl_zero(x)` | `countr_zero(x)` |
| Go | `bits.Len(x)` | `bits.OnesCount(x)` | `bits.LeadingZeros(x)` | `bits.TrailingZeros(x)` |

## 三、遍历集合

遍历集合中所有元素时，可以逐位判断，也可以不断取出最低位的 `1`。

### 逐位判断

```python
for i in range(n):
    if (s >> i) & 1:
        # i 在集合 s 中
        pass
```

### 不断提取最低位的 1

每次取出集合中最小的元素，然后删除这个元素，直到集合为空。

```python
t = s
while t:
    lb = t & -t
    i = lb.bit_length() - 1
    # i 在集合 s 中
    t ^= lb
```

## 四、枚举集合

### 4.1 枚举所有集合

枚举范围从 `0` 到 `2^n - 1`，每个整数都代表一个集合。

```python
for s in range(1 << n):
    # 处理子集 s
    pass
```

### 4.2 枚举非空子集

设集合为 `s`，从大到小枚举 `s` 的所有非空子集 `sub`：

```python
sub = s
while sub:
    # sub 是 s 的非空子集
    sub = (sub - 1) & s
```

为什么要写成 `sub = (sub - 1) & s`？

因为 `sub - 1` 会把最低位的 `1` 变成 `0`，并把它右边的所有 `0` 变成 `1`。再与原集合 `s` 做按位与，就能保证枚举出的集合始终是 `s` 的子集。

如果要包含空集，可以在循环结束后单独处理 `0`。

### 4.3 枚举子集（包含空集）

如果要从大到小枚举 `s` 的所有子集，可以这样写：

```python
sub = s
while True:
    # 处理 sub
    if sub == 0:
        break
    sub = (sub - 1) & s
```

### 4.4 枚举超集

如果全集为 `U`，要枚举所有包含 `s` 的超集，可以等价地枚举补集部分的子集。

```python
mask = (1 << n) - 1
extra = mask ^ s
sub = extra
while True:
    sup = s | sub
    # sup 是 s 的超集
    if sub == 0:
        break
    sub = (sub - 1) & extra
```
